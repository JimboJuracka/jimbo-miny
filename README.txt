HOW TO PLAY:

Select the difficulty when starting the game, using command ./game

-b BEGINNER - size 9x9   bombs: 10
-m MEDIUM   - size 16x16 bombs: 40
-e EXPERT   - size 16x30 bombs: 99
-c CUSTOM <height> <width> <number of bombs> (WARNING: Easily breakable)

You can always exit the game by closing the window, or pressing ESC key.

USED SDL LIBRARIES:
libsdl2-dev
libsdl2-image-2.0-0-dev
